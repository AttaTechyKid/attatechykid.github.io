How to make a Github Page (webpage using Github)

By AttaTechyKid

First create a respository and name it your username.github.io (ex. RGBgamerMsi.github.io)

Now click "upload an existing file"

Now upload files and commit changes. Make sure the file is index.html and etc.

Go to respository "Settings and then "Pages"

Now where it says Source, change that to Github Actions and put the custom domain as the same name of the respository.

If you get an error, switch it back to deploy from a branch.

You have just created a Github Page!
